# Press-It
Fast, simple memory game, made with the Arduino Uno/Nano.
Goal: Press the correct sequence of randomly flashing LEDs.

# Required components
| Part | Number |
|-----|-----|
|10 k Resistor | 4 |
|Color LED | 3-7|
| Button | 3-7|
| Arduino UNO | 1 |
